What's Alzheimer's

Alzheimer's disease is a progressive neurological disorder that affects memory and cognitive functions.

Common Symptoms

Memory Loss

Confusion

Difficulty in Communication

Early Stage

Memory lapses, difficulty finding words, and trouble organizing tasks.

In the early stage of Alzheimer's, individuals experience mild memory issues, forget names, and misplace objects.

Middle Stage

Increased confusion, difficulty recognizing people, and mood changes.

The middle stage is marked by greater memory loss, behavioral changes, and difficulties in everyday tasks.

Late Stage

Severe memory loss, difficulty speaking, and full-time care needed.

In the late stage, individuals require 24/7 care, experience severe cognitive decline, and struggle with communication.
